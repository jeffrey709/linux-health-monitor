# Linux Health Monitor

## Project Overview
This is a simple **Linux Health Monitor** script written in **Bash**.  
It automatically logs CPU, memory, and disk usage to a file for monitoring purposes.  

This project demonstrates:
- Linux command-line skills
- Bash scripting
- Basic automation concepts
- DevOps mindset

---

## Getting Started

### Prerequisites
- Linux system (Ubuntu, Debian, etc.)
- Bash shell (installed by default)
- Terminal access
